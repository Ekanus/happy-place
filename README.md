# Happy Place Website

Επίσημο website του Happy Place — Κέντρο Ειδικής Αγωγής, Μοίρες Κρήτης.

## Stack
Vanilla HTML, CSS, JS — hosted on Vercel.

## Environment Variables (Vercel Dashboard)
```
RESEND_API_KEY=your_key_here
```

## Pages
- `/` → Αρχική
- `/services` → Υπηρεσίες
- `/blog` → Blog
- `/contact` → Επικοινωνία

## Deploy
Push to GitHub → Vercel auto-deploys.

## Contact Form
Uses Resend API via `api/contact.js`.  
Set `RESEND_API_KEY` in Vercel dashboard.  
Verify `happyplace.gr` domain in Resend.

See `_skills/` folder for design system, animations, content, and stack reference.
